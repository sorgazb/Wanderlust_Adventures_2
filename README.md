# Wanderlust Adventures 2

![HTML5](https://img.shields.io/badge/HTML5-Maquetaci%C3%B3n-e34f26?style=for-the-badge&logo=html5)&nbsp;![CSS3](https://img.shields.io/badge/CSS3-Flexbox-1572b6?style=for-the-badge&logo=css3)&nbsp;![DIW - DAW](https://img.shields.io/badge/DIW-GS--DAW-1565c0?style=for-the-badge)

> **Wanderlust Adventures 2** es un sitio web estático de viajes desarrollado con **HTML5** y **CSS3** usando **Flexbox** como sistema de maquetación, como proyecto de la asignatura **Diseño de Interfaces Web (DIW)** del ciclo **GS-DAW** en el **IES Augustobriga**.

---

## 📋 Descripción

Evolución del proyecto Wanderlust Adventures aplicando **Flexbox** para la disposición y alineación de los elementos. Sus secciones principales incluyen:

- **Inicio** (`index.html`): página principal con presentación de la agencia.
- **Destinos** (`destinos.html`): catálogo de destinos disponibles.
- **Consejos** (`consejos.html`): guía de consejos para viajeros.
- **Contacto** (`contacto.html`): formulario de contacto con la agencia.

---

## 🏗️ Estructura del Proyecto

```txt
Wanderlust_Adventures_2/
├── index.html       # Página principal
├── destinos.html    # Página de destinos
├── consejos.html    # Página de consejos
├── contacto.html    # Página de contacto
├── css/             # Hojas de estilo CSS con Flexbox
└── img/             # Imágenes del sitio
```

---

## ⚙️ Ejecución

Clona el repositorio y abre directamente en el navegador:

```txt
git clone https://github.com/sorgazb/Wanderlust_Adventures_2.git
cd Wanderlust_Adventures_2
```

Abre `index.html` en tu navegador o usa la extensión **Live Server** de VS Code.
